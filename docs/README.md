## Review Basis

This review is based on direct inspection of:

- Root config and infrastructure files.
- API, worker, realtime, web dashboard, SDK, React bindings, and CLI code.
- SQL migrations and migration scripts.
- Existing production readiness notes.

The review intentionally distinguishes:

- Implemented and working behavior.
- Implemented but inconsistent behavior.
- Claimed behavior not yet implemented.
